# Social Follower Count Scraper

This is a node project that will scrape the follower count from Instagram and Twitter for specified accounts.

## Getting Started

- Run npm start 
```bash
  $ npm start
```

### Prerequisites

Node 6+ - [Website](https://nodejs.org/)

### Installing

- Run npm install
```bash
  $ npm i
```
